# World Cup 2026 Predictor

Focused prediction application with authentication, admin, and prediction league features.

## Live Demo
https://predictionfootballapp.netlify.app/

## Available Modules
- Authentication: Login / Signup / Profile / Settings
- Prediction: Dashboard, Matches, Leaderboard, Rooms, Rules, Admin
- Admin: Dashboard, Users, Content
- App shell: Sidebar + top navigation + dark mode toggle

## Development
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
```
